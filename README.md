# DNS Resolver

A DNS resolver server built with NestJS.

## Stack/Lib Choices

- PNPM (Package Manager)
- NestJS (NodeJS Framework)

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Using the Service

Call https://yourhostname.com/dns/:domain to get DNS information about the route.
